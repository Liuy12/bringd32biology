Introduction

RNA-seq Vis is a web application for analyzing RNA-seq datasets. Different from traditional application for processing RNA-seq data, by using multiple javascript library (see in Dependency section), RNA-seq Vis provides an easy way to generate dynamic visualization for RNA-seq pre-processing and differential expression analysis.

To use RNA-seq Vis, please firstly check the documentation on the left panel and make sure you follow every step by uploading your RNA-seq expression count file and experimental design file and setting all the options required for the analysis. After you submit all required inputs, the output figures will be automatically loaded into Charts subpanel on the left whenever the analysis has finished. You will have options to save selected figures as html report and/or pdf files. To get started with RNA-seq vis, please check the documentation panel on the left.

Dependency

RNA-seq Vis is constructed based on shiny framework from R studio. To generate interactive visualization, we incorporate multiple javascript libraries, including Polychart, Highcharts and NVD3. These javascript libraries are in turn based on D3 javascript library. To use these javascript libraries within shiny framework, we adopt rCharts package which is essentially an application programming interface (API) for these libraries. Additionally, we also utilize several packages from htmlwidgets project, including d3heatmap, DataTables, networkD3, MetricsGraphics and threejs. Google authentification functionality is adopted from googleAuthR package.

Differences with GenePattern

  • Input files format: GenePattern requires certain formats of input files. Users need to go through several steps just to get the correct format. However, RNA-seq Vis accepts input files in txt/csv format, which will save users some time in converting input files.
  • Analysis procedure: GenePattern uses cuffdiff for testing differential expression by default. Some popular R packages, such as DESeq, edgeR, limma, etc., are not incorporated. On the other side, RNA-seq Vis provides users with access to major differential expression testing R packages. We also include an R package called XBSeq that is developed in our own lab. Last but not least, users will have the freedom to change options for differential expression analysis to fit their own needs.
  • Dynamic visualization: RNA-seq Vis incorporated several popular javascript libraries for visualization which in turn depend on D3 javascript library. Users will have a different perspective of their datasets by interactive visualization.
