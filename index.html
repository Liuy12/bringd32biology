Introduction

Updated on September 29, 2016


Traditional RNA sequencing (RNA-seq) is a revolutionary technology that enables transcriptomic profiling in a systematic manner and has been widely adopted for genomic research during the past two decades. However, for traditional RNA-seq, profiles of gene expression levels are generated from bulk populations of millions of input cells, which means that the expression value for each gene is an average of expression levels across the bulk population. As a result, the underneath true signal of interest might be misrepresented. Thanks to the recent development of RNA-seq technology, we are now able to examine transcriptomic profile at single cell level. Up till now, there are still no web-based platforms for processing scRNA-seq datasets. Here we present an integrated web application, scRNA-seq Vis, for analyzing scRNA-seq datasets.

scRNA-seq Vis is a web application for analyzing single cell as well as general bulk RNA-seq datasets. Different from traditional application for processing RNA-seq data, by using multiple javascript library (see in Dependency section), scRNA-seq Vis provides an easy way to generate dynamic visualization for RNA-seq pre-processing and differential expression analysis.
