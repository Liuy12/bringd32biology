Introduction

scRNA-seq Vis is a web application for analyzing single cell as well as general bulk RNA-seq datasets. Different from traditional application for processing RNA-seq data, by using multiple javascript library (see in Dependency section), scRNA-seq Vis provides an easy way to generate dynamic visualization for RNA-seq pre-processing and differential expression analysis.

To use scRNA-seq Vis, please firstly check the documentation on the left panel and make sure you follow every step by uploading your RNA-seq expression count file and experimental design file and setting all the options required for the analysis. After you submit all required inputs, the output figures will be automatically loaded into Charts subpanel on the left whenever the analysis has finished. After the analysis has finished you can download the report files by pressing the download button in charts subpanel.

Dependency

scRNA-seq Vis is constructed based on shiny framework from R studio. To generate interactive visualization, we incorporate multiple javascript libraries, including Polychart, Highcharts and NVD3. These javascript libraries are in turn based on D3 javascript library. To use these javascript libraries within shiny framework, we adopt rCharts package which is essentially an application programming interface (API) for these libraries. Additionally, we also utilize several packages from htmlwidgets project, including d3heatmap, DataTables, networkD3, MetricsGraphics and threejs. Google authentification functionality is adopted from googleAuthR package. HTML report is generated using reveal.js that is incorporated in slidify package.

Difference in me

Differences with GenePattern and Galaxy

  • Input files format: GenePattern requires certain formats of input files. Users need to go through several steps just to get the correct format. However, scRNA-seq Vis accepts input files in txt/csv format, which will save users some time in converting input files.
  • Analysis procedure: GenePattern and Galaxy uses cuffdiff for testing differential expression by default. Some popular R packages, such as DESeq, edgeR, limma, etc., are not incorporated. For Galaxy, even through users can access some R packages through Galaxy tool shed, but many of the repositories within tool shed are not well mainteined. On the other side, scRNA-seq Vis provides users with access to most recent version of major differential expression testing R packages. We also include an R package called XBSeq that is developed in our own lab. Last but not least, users will have the freedom to change options for differential expression analysis to fit their own needs.
  • Single cell RNA-seq: We also provide support for single cell RNA-seq differential expression analysis including SCDE and
  • Dynamic visualization: scRNA-seq Vis incorporated several popular javascript libraries for visualization which in turn depend on D3 javascript library. Users will have a different perspective of their datasets by interactive visualization.

Differences with RNAseqGUI

  • R as prerequisite: RNAseqGUI is an package that depends on R, so users will need to install R first. In addition, RNAseqGUI is not deployed in either cran or Bioconductor. So users will have to install all the dependencies manually. However, for scRNA-seq Vis, users don't need to install any software locally. Neither do you need to have any programming experience with R.
  • Interactivity is always better: RNAseqGUI generally uses R base plotting system which only gives you dull and non-interactive figures. However, as mentions above, scRNA-seq Vis generates graphs with interactivity which will enable 'interpretation while plotting'.
