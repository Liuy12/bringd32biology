Case 1: Using DESeq2 to process bulk RNA-seq datasets

Datasets

The dataset used for this section comes from a mouse RNA-seq study, which contains 3 replicates of wild type mouse liver tissues (WT) and 3 replicates of Myc transgenic mouse liver tissues (MYC). The dataset is obtained from Gene Expression Omnibus (GSE61875). The expression matrix and experimental design file can be downloaded here: Expression dataset and design matrix.

Set options

Click on set input options subpanel of Start analysis panel on the left and firstly set experiment information shown below.

Choose DESeq2 as method for DE analysis and then upload Expression dataset and design matrix downloaded above.

Leave other options as default and click start analysis button in the bottom right box.

After clicking the start analysis button, you will see a progression bar on top of the page indicating current analysis process like shown below.

Click on the Charts subpanel on the left. All the figures will be loaded automatically once the analysis finished. And download button will also appear after the analysis finished.

Case 3: Identify highly variable genes (HVGs) for single-cell RNA-seq datasets

Datasets

The dataset used for this section consisted of 430 single glioblastoma cells isolated from 5 individual tumors. The dataset is obtained from Gene Expression Omnibus (GSE57872). The expression matrix file can be downloaded here: Expression dataset.

Set options

Click on set input options subpanel of Start analysis panel on the left and firstly set experiment information shown below. Remember to set experimental design option to 'not available'.

Choose BPSC as method for DE analysis and then upload Expression dataset downloaded above.

Leave other options as default and click start analysis button in the bottom right box.

After clicking the start analysis button, you will see a progression bar on top of the page indicating current analysis process like shown below.

Click on the Charts subpanel on the left. All the figures will be loaded automatically once the analysis finished. And download button will also appear after the analysis finished.
